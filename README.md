# Pipo & Preco template

Plantilla estatica para presentar una propuesta web a Pipo & Preco.

Preparada por Mario Rodriguez y Guillermo Campos.

## Abrir

Abre `index.html` en el navegador o lanza un servidor estatico desde esta carpeta.

## Fuente de contenido

- Ficha de MavelPoint: https://www.mavelpoint.com/es/artist/pipoypreco/overview
- Instagram publico indicado en la ficha: https://www.instagram.com/pipoyprecoshow
- Articulos enlazados desde la ficha de MavelPoint.
